# Lernprojekt: GitHub Grundlagen

Dieses Projekt dient dazu, die grundlegenden Funktionen von GitHub kennenzulernen:
- Repository erstellen
- Issues anlegen
- Milestones nutzen
- Project Boards verwenden
- Zusammenarbeit simulieren

## Projektstruktur
- /docs – Dokumentation
- /src – Beispielcode
- /tests – Testdateien

## Ziel
Das Ziel ist es, den Umgang mit GitHub zu üben und typische Workflows zu verstehen.
